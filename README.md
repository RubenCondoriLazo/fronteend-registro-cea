# SISTEMA EDUCATIVO RUBEN - Frontend CEA

Interfaz premium para el sistema académico CEA Bolivia (Humanística y Técnica).

## Tecnologías
- HTML5 / CSS3 (Glassmorphism Design)
- JavaScript Vanilla
- Outfit Google Font

## Despliegue en Netlify
1. Clonar el repositorio.
2. Subir a GitHub.
3. Conectar en Netlify (Build: None, Publish: `./`).

## Características
- Dashboard responsivo.
- Lógica de promedio para áreas humanísticas.
- Validación estricta para titulación técnica (5 módulos aprobados).
- Basado en el calendario de 200 días hábiles.
